Exercices Pratiques

Appliquez les concepts appris dans le cours à travers ces exercices pratiques.

Exercice 1: Création d'une page HTML

Créez une page HTML simple avec une structure de base (doctype, html, head, body) et ajoutez un titre et un paragraphe.

Exercice 2: Stylisation CSS

Ajoutez du CSS pour styliser votre page HTML. Changez la couleur du texte et ajoutez une marge autour du contenu.
